# Crop Yield Prediction

## Project Overview
This project predicts crop yield using Machine Learning.

## Algorithm Used
Random Forest Regressor

## Features Used
- Crop
- State
- Cost of Cultivation A2+FL
- Cost of Cultivation C2
- Cost of Production C2

## Target Variable
- Yield (Quintal/Hectare)

## Accuracy
94.63%

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Joblib
- VS Code

## Output
The model predicts crop yield and saves the trained model as crop_yield_model.pkl
